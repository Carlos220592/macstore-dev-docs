AlertifyJS es un librería de JavaScript diseñado para crear diálogos y notificaciones atractivas en
navegadores web. Este script facilita la implementación de elementos personalizados como alertas, confirmaciones y
prompts, así como notificaciones que son visualmente agradables y fáciles de personalizar.

## Uso

La implementación de Alertify JS es sumamente fácil, solo hay que tener en cuenta
algunos detalles de configuración de archivos como es el caso de las fuentes y los colores ya
que Alertify JS al ser una versión modificada para CX MacStore hace uso de estos archivos:
- **root.css**
- **fonts.css**

:::caution[Ejemplo de uso]
    Ejemplo de directorios <small><b>(Puede cambiar)</b></small>
    <FileTree>
        - public
            - css
                - root.css
                - fonts.css
            - vendor
                - alertify
                    - alertify.min.css
                    - alertify.min.js
        - index.html
    </FileTree>

    En el archivo **index.html** o donde se desea usar Alertify JS se debe primero cargar el
    archivo **root.css** y después **fonts.css**.

    ```html
        <!DOCTYPE>
        <html>
            <head>
                <link rel="stylesheet" href="/public/css/root.css" />
                <link rel="stylesheet" href="/public/css/fonts.css" />
                <link rel="stylesheet" href="/public/vendor/alertify/alertify.min.css" />
                <script src="/public/vendor/alertify/alertify.min.js"></script>
            </head>
        </html>
    ```

    **Nota:** Es recomendable hacer uso de layouts para evitar llamar más de una vez el mismo archivo.
:::
